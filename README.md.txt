# Customer Churn Prediction
A Python project to predict credit card customer churn for American Express, using Random Forest and Plotly.

## How to Run
1. Install Python and libraries: `pip install pandas numpy scikit-learn plotly`.
2. Download `Churn_Modelling.csv` from Kaggle.
3. Run `churn_prediction.py` to see accuracy and a churn drivers plot.

## Output
- Accuracy: ~85%.
- See `churn_plot.html` for visualization.
